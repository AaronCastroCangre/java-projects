## Todo List API

API REST completa para gestión de tareas orientada a principiantes.

**Tecnologías:** Java 17, Spring Boot 3, PostgreSQL, Docker, Swagger

**Inicio rápido:**

```bash
cd todo-list-api

# Linux/macOS
make all

# Windows
run.bat all
```
